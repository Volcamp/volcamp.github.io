A l'occasion d'une mise à jour de version, je tombe sur une incompatibilité avec un projet open source.

Un ticket existe déjà, mais aucun correctif n’est encore disponible.

Je décide de me lancer dans la contribution open source et de proposer une PR 💪.

Je vous propose un retour d'expérience sur cette aventure pleine de rebondissements.

Dans ce talk, nous aborderons notamment :

✅ Pourquoi et comment je me suis lancé dans cette contribution au projet ExternalDNS
🛠️ Les étapes concrètes : de la prise en main du projet jusqu’à la validation de la PR
⏳ Comment trouver le temps de contribuer quand on a déjà un quotidien chargé
📝 Le fonctionnement du Contributor License Agreement (CLA)
🤝 Le rôle des mainteneurs et le processus de code review
🌍 Comment la communauté open source facilite… ou complique parfois ! la collaboration *
💡 Quelques conseils pour débuter en douceur et se lancer quand on n’ose pas

Un talk qui je l'espere vous donnera envie de vous lancer dans la contribution open source 🙌.
