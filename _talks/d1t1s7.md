Une conférence dédiée aux développeurs souhaitant faire tourner leurs applications Java vers des architectures cloud natives. 
À travers des retours d’expériences concrets et des conseils pratiques, les intervenants partagent leurs points de vue pour containeriser, déployer et exploiter efficacement des applications Java dans  Kubernetes.  
La session met en lumière les enjeux de performance, de configuration et de layering dans un environnement distribué.
