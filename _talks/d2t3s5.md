Les assistants IA pour le code (Claude Code, Codex, OpenCode, GitHub Copilot, Cursor...) ont émergé et envahi l'écosystème de notre métier en développement logiciel.  Mais comment ça marche ? Quelle est la magie qui permet de créer et de modifier mon code, d'exécuter des commandes, etc...  
Je me posais la question. Alors j'ai creusé le sujet et j'ai développé mon propre assistant IA.... du moins, un début de mini assistant 😉.  
Je vous partagerai mon REX sur ce projet : les principes de base (LLM, tools, historique), des démos pour voir si ça marche (analyse, création de code, correction de bug, skill, MCP), les difficultés que j'ai rencontrées, les choses que j'ai apprises, et les bonnes surprises.
